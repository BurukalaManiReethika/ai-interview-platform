# AI Interview & Resume Intelligence Platform

AI-powered career preparation platform that:

- Analyzes resumes
- Calculates ATS score
- Extracts skills
- Generates interview questions
- Conducts AI mock interviews
- Evaluates answers
- Generates PDF reports
- Tracks analytics
- Provides leaderboard rankings

## Tech Stack

Backend:
- Flask
- SQLite
- SQLAlchemy

AI:
- Google Gemini

Frontend:
- HTML
- CSS
- JavaScript
- Bootstrap

Deployment:
- Render
